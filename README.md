# ❄️ Orb Balance Viewer

一款强大的VSCode插件，帮助你轻松管理多个Augment AI账号，实时查看余额，一键切换登录。

## ✨ 功能特性

- 🎫 **卡密登录** - 输入卡密即可自动获取并添加Augment账号
- 🔐 **Session登录** - 支持手动添加auth_session登录
- 💰 **余额显示** - 右下角状态栏实时显示当前账号余额
- 🔄 **自动刷新** - 每5分钟自动更新余额（可自定义）
- 👥 **多账号管理** - 支持添加、切换、删除多个账号
- 🚀 **一键登录** - 自动安装并登录Augment插件
- 📊 **详细信息** - 查看账号的完整Orb余额详情

## 📦 安装方法

### 方式1：从VSIX文件安装（推荐）

1. 下载最新的 `orb-balance-viewer-x.x.x.vsix` 文件
2. 打开VSCode
3. 按 `Ctrl+Shift+P` 打开命令面板
4. 输入 `Extensions: Install from VSIX...`
5. 选择下载的VSIX文件
6. 重启VSCode

### 方式2：从源码安装

```bash
# 克隆仓库
git clone <repository-url>
cd orb-balance-viewer

# 安装依赖
npm install

# 打包
npm run package

# 安装生成的VSIX文件
code --install-extension orb-balance-viewer-0.0.1.vsix
```

## 🚀 快速开始

### 第一步：打开侧边栏

点击VSCode左侧活动栏的 **❄️ 雪花图标**，打开Orb Balance侧边栏。

### 第二步：添加账号

#### 方式1：卡密登录（推荐）

1. 点击 **🎫 卡密** 标签
2. 输入你的卡密
3. 点击 **🎫 验证卡密** 按钮
4. 等待验证完成，账号自动添加

#### 方式2：Session登录

1. 点击 **🔐 Session** 标签
2. 粘贴你的 `auth_session` cookie值
3. 点击 **➕ 添加账号** 按钮
4. 等待账号信息加载完成

### 第三步：使用账号

- **查看余额** - 右下角状态栏显示当前账号余额
- **切换账号** - 点击账号卡片上的 **🔑 登录** 按钮
- **刷新余额** - 点击账号卡片上的 **🔄 刷新** 按钮
- **查看详情** - 点击 **📊 详情** 查看完整Orb余额信息
- **删除账号** - 点击 **🗑️ 删除** 移除账号

## 📖 详细功能说明

### 🎫 卡密登录

卡密登录是最简单的添加账号方式：

1. 从服务商处获取卡密
2. 在侧边栏的"卡密"标签页输入卡密
3. 点击"验证卡密"
4. 系统自动获取账号信息并添加

**优点**：
- 无需手动复制cookie
- 一键添加，简单快捷
- 自动获取完整账号信息

### 🔐 Session登录

适合高级用户手动添加账号：

1. 登录 [Augment Portal](https://portal.augmentcode.com)
2. 打开浏览器开发者工具（F12）
3. 进入 Application/Storage → Cookies
4. 复制 `auth_session` 的值
5. 粘贴到插件的"Session"标签页
6. 点击"添加账号"

### 💰 余额显示

- **状态栏显示**：右下角显示当前激活账号的余额
- **自动刷新**：默认每5分钟自动更新（可在设置中修改）
- **手动刷新**：点击状态栏或账号卡片的刷新按钮
- **详细信息**：点击"详情"查看完整的Orb余额数据

### 👥 多账号管理

- **添加账号**：支持卡密和Session两种方式
- **切换账号**：点击"登录"按钮切换到该账号
- **删除账号**：点击"删除"按钮移除账号
- **账号状态**：当前激活账号会显示"当前账号"标签

### 🚀 自动登录Augment

点击"登录"按钮后，插件会：

1. 检测是否安装了Augment插件
2. 如果未安装，自动安装修改版Augment插件
3. 自动注入登录凭证
4. 提示是否重新加载窗口以完成登录

## ⚙️ 配置选项

打开VSCode设置（`Ctrl+,`），搜索 `orbBalance`：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `orbBalance.refreshInterval` | 自动刷新间隔（毫秒） | 300000（5分钟） |
| `orbBalance.augmentAccounts` | 保存的账号列表 | [] |
| `orbBalance.activeAugmentAccount` | 当前激活的账号邮箱 | - |

## 🔧 常见问题

### Q: 卡密验证失败怎么办？

A: 请检查：
- 卡密是否正确
- 网络连接是否正常
- 卡密是否已过期

### Q: 状态栏不显示余额？

A: 请确保：
- 已添加至少一个账号
- 账号已激活（点击"登录"）
- 网络连接正常

### Q: 自动登录Augment失败？

A: 尝试：
1. 点击"重新加载窗口"
2. 手动重启VSCode
3. 检查Augment插件是否正确安装

### Q: 如何修改自动刷新间隔？

A:
1. 打开设置（`Ctrl+,`）
2. 搜索 `orbBalance.refreshInterval`
3. 修改为你想要的毫秒数（如60000=1分钟）
